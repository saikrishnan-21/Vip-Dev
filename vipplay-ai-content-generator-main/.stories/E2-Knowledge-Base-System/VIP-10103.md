# VIP-10103: Add Topic Source Manually

**Story ID**: VIP-10103
**Story Type**: Story
**Epic**: E2-Knowledge-Base-System
**Priority**: Medium
**Story Points**: 2
**Status**: To Do

## User Story

As a user, I want to add topics manually so that I can organize content by subject matter

## Acceptance Criteria

- [ ] **AC1**: User can enter topic name and description
- [ ] **AC2**: User can add multiple keywords (tags) to define the topic scope
- [ ] **AC3**: Topic is saved to MongoDB with validation
- [ ] **AC4**: Duplicate topics are prevented (same name for same user)
- [ ] **AC5**: Topic appears in Knowledge Base UI with active status

## Technical Details

**Frontend Components:**
- Update `AddTopicDialog` component with:
  - Topic Name (required, 2-100 chars)
  - Description (optional, textarea)
  - Keywords input with tag management (min 1 keyword required)
  - JWT authentication integration
  - Success/error toast notifications

**Backend API:**
- Create `POST /api/sources/topic` endpoint
- JWT authentication middleware
- MongoDB integration with duplicate name prevention
- Validation using `createTopicSourceSchema`

**Data Model:**
```typescript
{
  userId: ObjectId,
  type: 'topic',
  name: string,
  description?: string,
  status: 'active',
  topicKeywords: string[], // Min 1 keyword
  articlesCount: 0,
  createdAt: Date,
  updatedAt: Date
}
```

**Validation:**
- Topic name: 2-100 characters, required
- Description: optional
- Keywords: array of strings, minimum 1 required
- Prevent duplicate topic names per user
